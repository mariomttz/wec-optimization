# WEC Optimization

## 🛠️ Instalación 
1. Clonar el repositorio:  
```bash  
git clone https://github.com/mariomttz/wec-optimization.git

cd wave-energy-optimization  
```  

2. Instalar dependencias:  
```bash  
pip install -r requirements.txt  
```  

---

## 📊 Dataset  
El proyecto utiliza el dataset **"Large Scale Wave Energy Farm"** de UCI Machine Learning Repository:  
https://archive.ics.uci.edu/dataset/882/large-scale+wave+energy+farm  

---

## 📑 Artículo de referencia
Este trabajo se basa en el marco teórico propuesto en:  
**"Optimisation of large wave farms using a framework for coupling numerical wave propagation models with meta-heuristic optimisation algorithms"**  
(Neshat & Alexander, 2023)  
🔗 Enlace: https://www.semanticscholar.org/paper/Optimisation-of-large-wave-farms-using-a-framework-Neshat-Alexander/153a3eeff91e73eb45338719579a900972f7a9ca
